# Automação de Relatório de Vendas (Python)

Este projeto automatiza a geração de relatórios em PDF a partir de planilhas Excel de vendas.  
Ele lê os dados, calcula estatísticas e gera um gráfico automaticamente.

---

## Funcionalidades
- Leitura de planilha Excel
- Cálculo de total de vendas por produto
- Identificação do codigo do produto mais vendido
- Geração de gráfico e relatório PDF automático

---

## Tecnologias
- Python 3
- Pandas
- Matplotlib
- ReportLab
